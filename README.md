# 卉语传心科技有限公司

## 公司简介

卉语传心科技有限公司是由高中生自主发起、独立运营的科技创业团队，法人及董事长为李芝卉，核心成员均为在读高中生。团队秉持"卉心传语，以技为舟"理念，致力于用科技解决校园痛点、传递人文温度。

## 核心业务

- **实用产品研发**：桌面课表系统、AI寻课系统、轻语·虚拟伴侣
- **游戏开发**：《逃离CD模拟器》
- **公益项目**：《隐藏的话》

## 组织架构
