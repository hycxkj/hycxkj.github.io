## 三、详细总结

### （一）公司概况

卉语传心科技有限公司是**国内少有的高中生自主发起、独立运营**的科技类创业团队，**法人及董事长为李芝卉**，核心成员均为在读高中生，配备专业法务顾问保障合规运营。团队秉持 \\*\\*“卉心传语，以技为舟”\\*\\* 理念，以年轻视角解决真实需求，让高中生创意落地为实用产品。

### （二）组织架构

团队采用**轻量化、高效化**架构，共设 6 大部门：

1. **总经办**：核心决策中枢，统筹战略与运营

2. **科技研发事业部**：核心技术引擎，分网页、客户端、游戏 3 个开发组

3. **美术设计部**：视觉创意中心，负责全品类设计

4. **新媒体运营与宣传部**：负责品牌传播与用户连接

5. **法务部**：提供法律合规支持

6. **人事行政部**：负责后勤、人事与财务管理

### （三）核心业务（三大产品线）

1. **实用产品研发**

    - 桌面课表系统：轻量化课表工具，支持多端使用

    - AI 自主寻课系统：智能课程检索与教室定位

    - 轻语・你的虚拟伴侣：AI 情绪陪伴应用

2. **游戏开发**

    - 代表作品：《逃离 CD 模拟器》，校园背景解压模拟游戏

3. **公益类项目**

    - 待落地项目：《隐藏的话》，关注青少年心理健康

## 四、关键问题与答案

1. **问题 1：卉语传心的团队核心定位与最大特色是什么？**
**答案**：核心定位是**科技类创业团队**，最大特色是**由高中生自主发起、独立运营**，核心成员均为在读高中生，打破年龄对创业与创意的限制。

2. **问题 2：公司三大核心业务分别是什么，代表产品 / 项目有哪些？**
**答案**：三大核心业务为**实用产品研发、游戏开发、公益项目**；代表产品有桌面课表系统、AI 自主寻课系统、轻语・虚拟伴侣、《逃离 CD 模拟器》，待落地公益项目为《隐藏的话》。

3. **问题 3：团队组织架构的设计特点及核心决策部门是什么？**
**答案**：架构特点为**轻量化、高效化**，适配高中生团队运营；核心决策与统筹中枢是**总经办**，由董事长、总经理、技术 CEO 等组成，把控公司发展方向。
